## Don't Lie

If you list a skill, expect a 5-minute Q in the recruiter screen and a deeper one in the interview.

- Databricks Lakehouse badge → "I've done the Lakehouse Fundamentals course and built a small project on Community Edition" — defensible.
- Amplitude Foundations → "I built a funnel and a retention cohort on the sample dataset" — defensible.
- dbt Fundamentals certificate → "I built a small `staging → marts` project with tests and docs" — defensible.

Anything beyond what you actually did = fail interview, lose offer.
